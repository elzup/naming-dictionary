# aligned-naming

A reference for choosing variable name pairs that align in character length. Helps keep your code visually balanced.

## Aligned

Same length word pairs.

|   word | alt |
| -----: | --- |
| `head` |     |
| `tail` |     |

|    word | alt#1 |
| ------: | ----- |
| `black` | b     |
| `white` | w     |

|    word | alt |
| ------: | --- |
| `allow` |     |
| `block` |     |

|     word | alt#3 |
| -------: | ----- |
| `create` | -     |
| `update` | upd   |
| `delete` | del   |
